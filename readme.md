天大深夜雀食堂舞园 Bot - 提供对接 MinIO 的语录存储功能

仿 200 号龙 bot，提供如下功能

- 语录抽取功能，可以从 MinIO 中抽取语录，以及十连抽取
- 龙图抽取功能（）
- 存表情，可以将表情/动图的链接返回出来，以便下载
- 图片对称功能，可以将图片/动图进行上下左右对称
